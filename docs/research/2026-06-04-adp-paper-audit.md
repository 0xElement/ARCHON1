# ADP Paper Audit — arxiv 2601.19752 vs the Kurukshetra design

**Date:** 2026-06-04 · **Method:** 2 adversarial fresh-eyes lanes (missing-mechanisms + structural-blind-spots), each instructed to REFUTE the "nothing useful here" hypothesis. Anti-FOMO bar: a nice *name* for something we do is not a gap.
**Paper:** "Agentic Design Patterns: A System-Theoretic Framework" (Dao, Le, Lam, Le, Pham, O'Sullivan, Nguyen — UCC / VNU / IBM Research / Trinity). 5 subsystems (PG/RWM/AE/IAC/LA), 12 patterns, ReAct case study. **Conceptual only — zero benchmarks, authors admit it.**

## Headline
**10/12 patterns already covered or deliberately rejected** — often more rigorously than the paper (groupthink rejection, oracle-anchored anti-black-box, intrinsic-self-correction ban). But the paper's central thesis — *brittleness = missing TYPED subsystem boundaries* — exposed the design's biggest remaining structural blind spot, grounded in **two real production incidents**, not theory. The paper earned its read on exactly one axis, and that axis was our weakest.

## ADOPTED (folded into THE-FRAMEWORK "Typed-boundary refinements")

1. **Typed inter-phase envelope** *(high confidence — highest leverage)*. Ports are named by verb, not I/O type; every seam is file-path convention + regex. Evidence this is real, not academic: `VERDICT_RE` at the KRIPA→judge seam broke twice (2026-05-11/15), silently starving Phase 3.075+3.9 on a live example run; `finding-schema.js` documents 22 key signatures / 2 schema families in 48 findings; GATE-59/62 are point-patches locking single file paths after each break. `validateFinding` is auto-repair-never-reject = a coercion layer at ONE boundary, not a contract. Adopt: versioned envelope validated at every phase seam, fail-into-quarantine-LOUD, GATE-INTERPHASE-CONTRACT. Root-cause fix for the silent-drop / stale-file / field-name-mismatch incident class (same family as the 2026-04-27 false early-exit and the Gulf Oil dashboard bug).
2. **Typed Feedback record for OBSERVE** *(medium)*. The learning loop must consume typed episode records emitted at phase completion, not grep polluted JSONL (`trajectory-observer.js` documents the shell-echo pollution + schema_version workaround). Otherwise the adaptive shell inherits the untyped-boundary brittleness before the loop even ships.
3. **A4 — bounded re-prioritization wave** *(medium)*. Spawn set today is frozen at recon (A3 only subtracts; `/goal` deepens one objective). One bounded re-spawn wave (≤1/dispatch) triggered ONLY by oracle-verified external signals (confirmed finding reveals new in-scope surface). Passes the design's own "is there external feedback?" discriminator. For a security tool the highest-value finding is often the unpredicted one.
4. **Unified PreToolUse action-gate** *(borderline — cheap unification, not a new pattern)*. Generalize quarantine + GATE-EGRESS-DENY + active-PoC gating into one native PreToolUse hook with a per-squad conduct rubric (destructive-PoC shapes, exfil-shaped egress); deny → escalate. Today no live control can block a harmful-but-in-scope ACTION mid-flight — DHARMA grades after the act.

## CONFIRMED COVERED / DELIBERATELY REJECTED (the other 10)
- **Integrator** → data-fence + typed-untrusted-object + Phase 3.07 + validateFinding. **Retriever** → remember port + md-files + Dreaming (vector-DB = rejected implementation of the same pattern). **Recorder** → WAL + checkpoints + resumeFromRunId; reasoning deliberately stateless (12-Factor) — conscious choice, not gap. **Planner** → recon classify-and-route + squad topology. **Deliberator** → REJECTED framing (LATS/ToT/MCTS on the NOT-building list; ICLR-2024 intrinsic-self-correction ban); substitute = model routing + A1 bounded retry. **Executor** → Agent SDK structured stream (kills stdout-parsing silent drops). **Tool Use** → ports/adapters + MCP oracles + env-allowlist (stronger: paper omits hostile-tool defense). **Coordinator** → A2A rule-based handoff; live peer debate deliberately rejected (groupthink = 65% of debate errors). **Reflector** → DISTILL stage (GEPA/DSPy causal analysis). **Skill Build** → Voyager-style SKILL.md induction, 2-tap gated.
- **Strategy-vs-Knowledge update split** → adds nothing; our 4 improvement types (memory/skill/prompt/capability) subsume it and are friction-keyed, which the paper's split isn't.
- **Adaptive-shell-outside-the-core** → ours is STRONGER: safety perimeter structurally bars the improver from editing judge/eval/reward — the paper only says LA "sends updates."

## Bottom line
A vocabulary paper with no evidence — but used as an audit checklist, it caught the one structural debt class our 93 gates only ever point-patched. Net adds: 2 high-leverage typed-contract items + 1 bounded autonomy item + 1 cheap hook unification. Everything else: corroboration that the design is ahead.
