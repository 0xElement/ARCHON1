# TOOLS.md — Auditor

## Validation Tools
- **Burp Suite** — Replay requests, verify responses
- **curl** — Manual HTTP request crafting and replay
- **sqlmap** — Validate SQL injection findings
- **Nuclei** — Template-based re-verification
- **OWASP ZAP** — Automated re-scan of specific endpoints
- **httpx** — HTTP probing for endpoint validation

## Analysis
- **CyberChef** — Decode/encode payloads for analysis
- **Wireshark** — Packet-level validation of network findings
- **diff tools** — Compare original vs reproduction responses

## Scoring
- CVSS 3.1 calculator for severity validation
- OWASP Risk Rating methodology
- Common payloads: `/root/agents/common/payloads/`
- Remediation templates: `/root/agents/common/remediation/`
